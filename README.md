# 宠迹 - 宠物生活伴侣

一款融合宠物分享、健康管理、习惯打卡的小程序，定位为"宠物的生活记录工具+轻社交社区"。

## 项目概述

- **项目名称**：宠迹 / 宠叮当 / 客事
- **产品定位**：宠物的生活记录工具+轻社交社区
- **目标用户**：20-35岁，女性偏多，有记录和分享习惯
- **核心价值**：帮助宠主科学记录宠物健康、养成良好养宠习惯，同时通过轻松的内容分享获得情感满足

#### 数据库设计

**核心表结构**：

1. **users** - 用户表
   - openid, unionid, nickname, avatar, phone, gender

2. **pets** - 宠物表
   - user_id, name, breed, gender, birthday, avatar, weight

3. **checkin_items** - 打卡项表
   - user_id, name, icon, sort_order

4. **checkin_records** - 打卡记录表
   - user_id, pet_id, item_id, record_date

5. **weight_records** - 体重记录表
   - pet_id, weight, record_date

6. **posts** - 动态表
   - user_id, pet_id, content, images

7. **likes** - 点赞表
   - user_id, post_id

8. **comments** - 评论表
   - user_id, post_id, content, parent_id

9. **achievements** - 成就表
   - name, description, icon, condition_type, condition_value

10. **user_achievements** - 用户成就表
    - user_id, achievement_id, unlocked_at

11. **vaccine_reminders** - 疫苗提醒表
    - pet_id, vaccine_name, next_date, status

12. **parasite_reminders** - 驱虫提醒表
    - pet_id, type, next_date, status

### 安全架构

#### 1. 认证授权

```
用户登录
   ↓
前端获取OpenID
   ↓
后端验证/创建用户
   ↓
生成JWT Token (7天有效)
   ↓
前端存储Token
   ↓
后续请求携带Token
   ↓
JwtAuthenticationFilter验证
   ↓
Spring Security授权
```

#### 2. API安全

- **JWT Token认证**：所有需要认证的接口都要求携带有效的JWT Token
- **CORS配置**：限制跨域请求来源
- **防SQL注入**：使用MyBatis-Plus参数化查询
- **防XSS攻击**：输入输出转义
- **防CSRF攻击**：使用Token机制

#### 3. 数据安全

- **敏感数据加密**：密码使用BCrypt加密（预留）
- **HTTPS传输**：生产环境强制HTTPS
- **HTTPS证书**：使用Let's Encrypt免费证书
- **文件上传安全**：文件类型验证、大小限制

### 性能优化方案

#### 1. 缓存策略

```
热点数据缓存
├─→ 用户信息 (Redis TTL: 1小时)
├─→ 宠物信息 (Redis TTL: 1小时)
├─→ 打卡统计 (Redis TTL: 5分钟)
├─→ 动态列表 (Redis TTL: 10分钟)
└─→ 成就信息 (Redis TTL: 30分钟)
```

#### 2. 数据库优化

- **索引优化**：为常用查询字段添加索引
- **分页查询**：列表接口使用分页
- **读写分离**：主从复制（预留）
- **慢查询优化**：定期分析慢查询日志

#### 3. 文件存储优化

- **CDN加速**：图片使用阿里云OSS + CDN
- **图片压缩**：上传前压缩图片

#### 4. 接口优化

- **响应压缩**：Nginx Gzip压缩
- **接口限流**：使用Redis实现限流
- **异步处理**：耗时操作异步化

### 技术选型理由

#### 为什么选择Java 21？

- **性能优势**：虚拟线程、更好的并发性能
- **新特性**：Sealed Classes、Pattern Matching
- **生态成熟**：Spring Boot生态完善
- **长期支持**：Oracle提供长期支持

#### 为什么选择Spring Boot？

- **快速开发**：约定优于配置
- **生态丰富**：丰富的Starter依赖
- **社区强大**：活跃的开源社区
- **企业级**：广泛应用于企业项目

#### 为什么选择MyBatis-Plus？

- **简化开发**：自动CRUD操作
- **功能强大**：内置分页、条件构造器
- **性能优秀**：高性能ORM框架
- **易于维护**：清晰的代码结构

#### 为什么选择阿里云OSS？

- **高可靠**：99.995%可用性
- **高扩展**：弹性伸缩
- **高性价比**：按量付费
- **CDN集成**：全球加速
