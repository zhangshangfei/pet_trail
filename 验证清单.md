## ⚠️ 需要注意的问题

### 1. 前端环境配置
`env.js` 使用了 uni-app 的条件编译语法，需要确保：
- 在 `manifest.json` 中配置正确的环境变量
- H5 和小程序环境使用不同的 API 地址

### 2. 后端依赖
需要确保 `pom.xml` 中有以下依赖：
- ✅ Spring Boot Starter Security
- ✅ Spring Boot Starter Validation
- ✅ JWT (io.jsonwebtoken)
- ✅ Lombok

### 3. 登录页面路由
`pages.json` 已添加登录页配置，但需要确保：
- 小程序中设置为 `navigationStyle: custom`（自定义导航栏）
- 首次启动时跳转到登录页
