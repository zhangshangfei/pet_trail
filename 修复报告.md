## 启动说明

### 后端启动
```bash
cd backend

# 1. 复制环境变量文件
cp ../.env.example .env

# 2. 修改 .env 中的配置
# - WECHAT_MINIAPP_APP_SECRET
# - JWT_SECRET
# - ALIYUN_OSS_ACCESS_KEY_ID
# - ALIYUN_OSS_ACCESS_KEY_SECRET

# 3. 启动后端
mvn spring-boot:run
```

### 前端启动
```bash
cd frontend

# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问 http://localhost:8080
```

## 安全建议

### 生产环境必须修改的配置：
1. **JWT_SECRET** - 使用 `openssl rand -base64 32` 生成
2. **WECHAT_MINIAPP_APP_SECRET** - 在微信公众平台获取
3. **ALIYUN_OSS_ACCESS_KEY_ID/SECRET** - 在阿里云控制台获取
4. **MYSQL_PASSWORD** - 使用强密码

### 建议添加的安全措施：
1. HTTPS 证书配置
2. 请求频率限制
3. SQL 注入防护验证
4. XSS 防护
5. CSRF 防护
