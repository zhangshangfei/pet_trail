# 宠迹项目启动说明

## 项目目录
- 前端：`G:\openclaw_project\pet-trail\frontend`
- 后端：`G:\openclaw_project\pet-trail\backend`

---

## 前端启动步骤

### 1. 进入前端目录
```powershell
cd G:\openclaw_project\pet-trail\frontend
```

### 2. 安装依赖（首次运行）
```powershell
npm install
```

### 3. 启动开发服务器
```powershell
npm run dev
```

### 4. 访问前端页面
打开浏览器访问：http://localhost:8080

---

## 后端启动步骤

### 1. 进入后端目录
```powershell
cd G:\openclaw_project\pet-trail\backend
```

### 2. 启动Spring Boot应用
```powershell
mvn spring-boot:run
```

### 3. 访问后端接口
- API地址：http://localhost:8080
- 健康检查：http://localhost:8080/api/health

## 数据库配置

后端配置文件：`backend/src/main/resources/application.yml`

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/pet_trail?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: root123654

  data:
    redis:
      host: localhost
      port: 6379
```

确保MySQL和Redis服务已启动。

## 故障排查

### 前端启动失败
1. 检查Node.js版本：`node -v`
2. 清除缓存重新安装：`rm -rf node_modules package-lock.json && npm install`
3. 检查端口占用：`netstat -ano | findstr "8080"`

### 后端启动失败
1. 检查Java版本：`java -v`
2. 检查Maven版本：`mvn -v`
3. 检查数据库连接：确保MySQL和Redis运行中
4. 查看详细日志：关注控制台输出
