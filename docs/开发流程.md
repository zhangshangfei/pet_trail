## 开发流程

### 第一步：环境准备（1-2小时）

#### 1.1 安装必要软件
```bash
# 检查Java版本
java -version  # 需要21+

# 检查Maven版本
mvn -version

# 检查MySQL
mysql --version

# 检查Redis
redis-cli --version

# 检查Docker
docker --version
```

#### 1.2 克隆/进入项目
```bash
cd G:\openclaw_project\pet-trail
```

#### 1.3 配置环境变量
创建 `backend/.env` 文件：

```env
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=root
DB_NAME=pet_trail

# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379

# 阿里云OSS配置
ALIYUN_OSS_ACCESS_KEY_ID=your-access-key-id
ALIYUN_OSS_ACCESS_KEY_SECRET=your-access-key-secret

# JWT配置
JWT_SECRET=your-secret-key-at-least-256-bits-long

# 前端URL
FRONTEND_URL=http://localhost:5173
```

### 第二步：初始化数据库（30分钟）

#### 2.1 创建数据库
```bash
# 方法1：使用命令行
mysql -u root -p
CREATE DATABASE pet_trail DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit;
```

#### 2.2 执行初始化脚本
```bash
cd backend
mysql -u root -p pet_trail < src/main/resources/init.sql
```

#### 2.3 验证数据库
```bash
mysql -u root -p pet_trail
SHOW TABLES;
```

应该看到以下表：
- users
- pets
- checkin_items
- checkin_records
- weight_records
- posts
- likes
- comments
- achievements
- user_achievements
- vaccine_reminders
- parasite_reminders

### 第三步：启动后端服务（30分钟）

#### 3.1 编译项目
```bash
cd backend
mvn clean install
```

#### 3.2 启动后端
```bash
mvn spring-boot:run
```

#### 3.3 验证服务
```bash
# 健康检查
curl http://localhost:8080/api/health

# 访问Swagger文档
# 浏览器打开：http://localhost:8080/swagger-ui/index.html
```

#### 3.4 测试文件上传
```bash
curl -X POST http://localhost:8080/api/upload \
  -F "file=@test.jpg"
```

### 第四步：启动前端服务（30分钟）

#### 4.1 安装依赖
```bash
cd frontend
npm install
```

#### 4.2 配置API地址
编辑 `frontend/config/index.js`：

```javascript
export default {
  baseUrl: 'http://localhost:8080',
  ossUrl: 'http://localhost:8080'
}
```

#### 4.3 启动开发服务器
```bash
npm run dev
```

#### 4.4 验证前端
浏览器访问：http://localhost:5173

## 常用命令

### 后端开发
```bash
# 编译
mvn clean compile

# 运行
mvn spring-boot:run

# 打包
mvn clean package

# 运行jar
java -jar target/pet-trail-backend-1.0.0.jar

# 运行测试
mvn test

# 生成文档
mvn javadoc:javadoc
```

### 前端开发
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 运行测试
npm run test
```

### Docker
```bash
# 构建镜像
docker build -t pet-trail-backend .

# 启动服务
docker-compose up -d

# 查看日志
docker-compose logs -f backend

# 停止服务
docker-compose down

# 清理数据
docker-compose down -v
```
