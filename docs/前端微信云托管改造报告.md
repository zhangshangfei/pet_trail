## 技术要点

### 1. 环境判断逻辑

使用 uni-app 的条件编译 + 运行时判断：

```javascript
// #ifdef MP-WEIXIN
if (USE_CLOUD && wx.cloud) {
  return cloudRequest(options)
}
// #endif
return httpRequest(options)
```

**优势**：
- 编译时即可确定代码分支
- 运行时再次检查云开发可用性
- 支持降级，提高可用性

### 2. 云托管请求格式

```javascript
wx.cloud.callContainer({
  config: {
    env: 'prod-3gyc3tpg28270da6'  // 云托管环境 ID
  },
  path: '/api/users/login',        // API 路径
  header: {
    'X-WX-SERVICE': 'springboot-4fyd',  // 必需：服务名称
    'Content-Type': 'application/json',
    'Authorization': 'Bearer <token>'
  },
  method: 'POST',
  data: { code: 'xxx' }
})
```

**关键点**：
- `X-WX-SERVICE` 请求头必须设置
- `config.env` 指定云托管环境
- `path` 是 API 路径，不包含域名

### 3. 日志输出

请求日志：
```javascript
console.log('云托管请求:', {
  env: CLOUD_CONFIG.env,
  service: CLOUD_CONFIG.service,
  path: options.url,
  method: options.method || 'GET'
})
```

响应日志：
```javascript
console.log('云托管响应:', {
  statusCode: res.statusCode,
  data: res.data
})
```

**优势**：
- 便于调试
- 快速定位问题
- 可在微信开发者工具控制台查看

### 4. 降级策略

当云开发不可用时，自动降级为普通 HTTP 请求：

```javascript
if (!wx.cloud) {
  console.error('当前环境不支持微信云开发，降级为普通 HTTP 请求')
  return httpRequest(options)
}
```

**适用场景**：
- 本地开发调试
- 云开发服务异常
- 网络问题导致云开发不可用

## 部署流程

### 前端部署

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 构建微信小程序
npm run build:mp-weixin

# 4. 使用微信开发者工具打开 dist/build/mp-weixin/
# 5. 上传代码到微信公众平台
```

### 后端部署

1. 构建 Docker 镜像：
   ```bash
   docker build -t pet-trail-backend:latest .
   ```

2. 推送镜像到云托管：
   - 方式 A：使用微信开发者工具
   - 方式 B：使用云托管 CLI

3. 在云托管控制台部署服务：
   - 配置环境变量
   - 配置端口（8080）
   - 点击部署

## 注意事项

### 1. 云开发初始化

确保 `main.js` 中的云开发初始化代码在应用启动时执行：

```javascript
// #ifdef MP-WEIXIN
if (wx.cloud) {
  wx.cloud.init({
    env: config.VITE_CLOUD_CONFIG.env,
    traceUser: true
  })
}
// #endif
```

### 2. AppID 配置

确保 `project.config.json` 和 `manifest.json` 中的 `appid` 一致：
```json
{
  "appid": "wxb210d75568abe1f7"
}
```

### 3. 服务名称

`X-WX-SERVICE` 请求头必须正确设置，否则云托管无法路由请求。

### 4. 本地开发

本地开发时，可以临时修改 `env.js` 使用普通 HTTP 请求：
```javascript
// #ifdef MP-WEIXIN
return 'http://localhost:8080'  // 临时改为本地
// #endif
```
