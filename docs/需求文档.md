> 版本：v2.1.0 | 更新日期：2026-05-20

---

## 一、产品概述

### 1.1 产品定位

**宠迹**是一款面向宠物主人的**宠物生活记录工具 + 轻社交社区**应用，以微信小程序为主要载体，同时支持 H5 访问。

### 1.2 目标用户

- **核心用户**：20-35 岁宠物主人，女性偏多
- **用户痛点**：
  - 宠物健康记录分散，疫苗/驱虫容易遗忘
  - 缺乏科学养宠的记录和提醒工具
  - 养宠生活中缺少同好社交圈
  - 宠物用品选择困难，缺少专业推荐

### 1.3 核心价值

1. **科学记录**：系统化记录宠物健康、体重、疫苗、驱虫等数据
2. **习惯养成**：通过打卡系统和提醒功能，帮助养成良好养宠习惯
3. **情感满足**：通过动态分享、互动评论获得社交认同
4. **智能分析**：AI 驱动的宠物健康分析，提供科学养宠建议

### 1.4 产品形态

| 端 | 技术栈 | 说明 |
|---|--------|------|
| 微信小程序 | uni-app + Vue 3 | 用户端主要入口 |
| H5 页面 | uni-app + Vue 3 | 备用访问方式 |
| 管理后台 | Vue 3 + Element Plus | 运营管理后台 |
