# 启动文件说明

## Java 版本启动文件

### 1. PetTrailBackendApplication.java
**位置**: `backend/src/main/java/com/pettrail/pettrailbackend/PetTrailBackendApplication.java`

**作用**: Spring Boot 应用启动类

```java
@SpringBootApplication
@MapperScan("com.pettrail.pettrailbackend.mapper")
public class PetTrailBackendApplication {

    public static void main(String[] args) {
        SpringApplication.run(PetTrailBackendApplication.class, args);
    }
}
```

**说明**:
- `@SpringBootApplication`: 启用 Spring Boot 自动配置
- `@MapperScan`: 扫描 MyBatis Mapper 接口
- `main()`: 应用程序入口，启动 Spring Boot 容器

---

### 2. AppModule.java
**位置**: `backend/src/main/java/com/pettrail/pettrailbackend/AppModule.java`

**作用**: 模块配置类

```java
@Configuration
@MapperScan("com.pettrail.pettrailbackend.mapper")
public class AppModule {

    @Bean
    public PasswordEncoder passwordEncoder() {
        return new BCryptPasswordEncoder();
    }
}
```

**说明**:
- `@Configuration`: 标识为配置类
- `passwordEncoder()`: 提供 BCrypt 密码编码器 Bean（预留，当前使用 JWT）

## 文件对比

| 功能 | Node.js (NestJS) | Java (Spring Boot) |
|------|------------------|-------------------|
| 应用入口 | `main.ts` | `PetTrailBackendApplication.java` |
| 模块配置 | `app.module.ts` | `AppModule.java` |
| 启动方式 | `node main.ts` | `java -jar app.jar` 或 `mvn spring-boot:run` |
| 配置文件 | `app.module.ts` + `main.ts` | `application.yml` + `AppModule.java` |

## 启动方式

### 方式 1: 使用 Maven
```bash
cd backend
mvn spring-boot:run
```

### 方式 2: 打包后运行
```bash
# 1. 编译打包
cd backend
mvn clean package -DskipTests

# 2. 运行 JAR
java -jar target/pet-trail-backend-1.0.0.jar
```

### 方式 3: 使用 Docker
```bash
docker-compose up -d backend
```

## 配置文件说明

### application.yml
**位置**: `backend/src/main/resources/application.yml`

**作用**: Spring Boot 应用配置

**重要配置**:
```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/pet_trail
    username: root
    password: root

aliyun:
  oss:
    endpoint: oss-cn-hangzhou.aliyuncs.com
    accessKeyId: ${ALIYUN_OSS_ACCESS_KEY_ID}
    accessKeySecret: ${ALIYUN_OSS_ACCESS_KEY_SECRET}
    bucketName: pet-trail-bucket

jwt:
  secret: ${JWT_SECRET}
  expiration: 7d
```
