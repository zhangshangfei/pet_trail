# 用户模块说明

## 概述

用户模块负责处理用户相关的所有操作，包括登录、注册、资料查询和更新。

## 文件结构

```
backend/src/main/java/com/pettrail/pettrailbackend/
├── entity/
│   └── User.java                    # 用户实体类
├── mapper/
│   └── UserMapper.java              # MyBatis-Plus Mapper
├── controller/
│   └── UserController.java          # 用户控制器
├── service/
│   └── UserService.java             # 用户服务
└── AppModule.java                   # 模块配置
```

## 核心功能

### 1. 用户登录
- 根据微信OpenID查询用户
- 如果用户不存在，自动创建新用户
- 返回JWT Token

### 2. 用户注册
- 使用OpenID或UnionID注册
- 如果用户已存在，返回错误
- 自动生成Token

### 3. 获取用户资料
- 需要JWT认证
- 返回完整的用户信息

### 4. 更新用户资料
- 需要JWT认证
- 支持部分字段更新
- 返回更新后的用户信息

## JWT 认证流程

1. **登录/注册**：
   - 前端传递OpenID
   - 后端查询或创建用户
   - 生成JWT Token
   - 返回Token给前端

2. **后续请求**：
   - 前端在请求头携带Token：`Authorization: Bearer {token}`
   - JwtAuthenticationFilter拦截请求
   - 验证Token有效性
   - 提取userId并设置到SecurityContext
   - Controller可以获取当前用户

## 数据库表结构

```sql
CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,
  openid VARCHAR(100) UNIQUE NOT NULL,
  unionid VARCHAR(100) UNIQUE,
  nickname VARCHAR(50),
  avatar VARCHAR(255),
  phone VARCHAR(20),
  gender TINYINT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_openid (openid),
  INDEX idx_unionid (unionid)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 安全考虑

1. **JWT Secret**: 在`.env`文件中配置，至少256位
2. **Token过期**: 默认7天
3. **密码加密**: 使用BCrypt（预留）
4. **SQL注入**: 使用MyBatis-Plus参数化查询
5. **权限控制**: 使用Spring Security + JWT
