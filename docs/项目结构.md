## 后端模块说明

### 1. 配置模块 (config)

- **OssConfig**: 阿里云OSS配置类
- **SecurityConfig**: Spring Security配置，包含JWT认证
- **SwaggerConfig**: Swagger API文档配置

### 2. 控制器模块 (controller)

- **HealthController**: 健康检查接口
- **UploadController**: 文件上传接口（OSS）

### 3. 服务模块 (service)

- **OssService**: OSS文件上传、删除等操作

### 4. 安全模块 (security)

- **JwtAuthenticationFilter**: JWT过滤器，拦截请求并验证Token

### 5. 工具模块 (util)

- **JwtUtil**: JWT生成、验证、解析工具类

## 前端模块说明

### 1. 页面 (pages)

- 用户相关页面（登录、注册、个人中心）
- 宠物管理页面
- 打卡页面
- 社区页面
- 数据看板页面

### 2. 组件 (components)

- 通用组件（按钮、输入框、卡片等）
- 业务组件（打卡组件、成就组件等）

### 3. API (api)

- 封装后端API调用
- 统一错误处理

### 4. Store (store)

- 用户状态管理
- 宠物状态管理
- 打卡状态管理

## 数据库模块说明

### 核心表

1. **users**: 用户表
2. **pets**: 宠物表
3. **checkin_items**: 打卡项表
4. **checkin_records**: 打卡记录表
5. **weight_records**: 体重记录表
6. **posts**: 动态表
7. **likes**: 点赞表
8. **comments**: 评论表
9. **achievements**: 成就表
10. **user_achievements**: 用户成就表
11. **vaccine_reminders**: 疫苗提醒表
12. **parasite_reminders**: 驱虫提醒表

## 开发规范

### 后端规范

1. 使用Lombok简化代码
2. 使用MyBatis-Plus简化CRUD操作
3. 使用Swagger生成API文档
4. 使用JWT进行用户认证
5. 使用Redis做缓存
6. 使用阿里云OSS存储文件

### 前端规范

1. 使用Uni-app开发小程序
2. 使用uView UI组件库
3. 使用Pinia做状态管理
4. 使用ECharts做图表
5. 组件化开发
6. API统一封装

### 代码规范

1. 后端使用阿里巴巴Java开发手册
2. 前端使用ESLint + Prettier
3. 统一使用Git进行版本控制
4. 使用Git Flow工作流

## 部署架构

```
┌─────────────────────────────────────────┐
│           阿里云 ECS                    │
│  ┌─────────────────────────────────┐   │
│  │  Nginx (反向代理 + 静态资源)      │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │  Spring Boot 应用                │   │
│  │  - REST API                      │   │
│  │  - WebSocket (实时互动)          │   │
│  │  - 任务调度 (提醒)               │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │  MySQL (主库)                    │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │  Redis (缓存 + 会话)             │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
                  │
                  │ OSS
                  ↓
         ┌─────────────────┐
         │  阿里云 OSS     │
         │  (文件存储)     │
         └─────────────────┘
```

## 扩展模块

### 可选功能

1. **消息队列**: 使用RocketMQ或RabbitMQ实现异步处理
2. **搜索引擎**: 使用Elasticsearch实现全文搜索
3. **消息推送**: 使用极光推送、个推等实现消息推送
4. **监控告警**: 使用Prometheus + Grafana实现监控
5. **日志分析**: 使用ELK Stack实现日志分析
6. **定时任务**: 使用Spring Task或XXL-Job实现定时任务
