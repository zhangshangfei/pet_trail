# Tabbar 图标说明

## 所需图标列表

请在以下网站下载免费图标，或使用设计工具创建：

### 推荐图标资源
1. [IconFont](https://www.iconfont.cn/) - 阿里巴巴矢量图标库
2. [Flaticon](https://www.flaticon.com/) - 免费 PNG 图标
3. [IconPark](https://iconpark.oceanengine.com/) - 字节开源图标库

### 所需图标

| 名称 | 尺寸 | 用途 | 建议样式 |
|------|------|------|----------|
| dashboard.png | 81x81 | 看板（未选中） | 灰色图标 |

## 临时解决方案

如果暂时没有图标，可以：
1. 使用 uni-app 内置图标
2. 使用文字代替
3. 使用 base64 编码的简单图形

## 图标设计要求

- 格式：PNG
- 尺寸：81x81 像素（实际显示 48rpx）
- 背景：透明
- 风格：简洁、线性或面性图标
